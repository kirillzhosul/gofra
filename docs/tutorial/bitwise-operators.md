# Bitwise operators

# Shift left and right (<<, >>)

Logical shift right and left
Example
```gofra
2 3 << // logical shift 2 by 2 bits to the left
2 3 >> // logical shift 2 by 3 bits to the right
```

# AND (&)
```gofra
2 3 & // 0b10 & 0b11 -> 0b10 -> 2
```

# XOR (^)
```gofra
2 3 ^ // 0b10 ^0b11 -> 0b01 -> 1
```
